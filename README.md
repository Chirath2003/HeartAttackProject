# ❤️ Heart Attack Prediction System

A Machine Learning project that predicts the risk of heart attacks using patient health and lifestyle data.

## 📌 Project Overview

This project analyses a heart attack dataset and uses Machine Learning techniques to predict whether a patient is at risk of experiencing a heart attack.

The project covers:

* Data collection and preprocessing
* Exploratory Data Analysis (EDA)
* Feature engineering
* Machine Learning model training
* Model evaluation
* Prediction generation

## 📂 Dataset

Dataset used:

* `heart_attack_prediction_indonesia.csv`

The dataset contains various health-related features such as:

* Age
* Gender
* Blood Pressure
* Cholesterol Level
* BMI
* Smoking Status
* Diabetes Status
* Heart Rate
* Physical Activity
* Other clinical indicators

### Target Variable

| Value | Meaning         |
| ----- | --------------- |
| 0     | No Heart Attack |
| 1     | Heart Attack    |

## ⚙ Installation

Clone the repository:

```bash
git clone https://github.com/Chirath2003/HeartAttackProject.git
```

Move into the project directory:

```bash
cd HeartAttackProject
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Or:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn
```

## 🚀 Running the Project

Run dataset analysis:

```bash
cd Dataset
python dataset_analysis.py
```
